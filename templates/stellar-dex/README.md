# Stellar DEX Template

A decentralized exchange template built on the Stellar blockchain.

## Features

- Token swapping on Stellar network
- Real-time price feeds
- Transaction history
- Wallet integration
- Customizable branding

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Configuration

This template is configured via environment variables. See `.env.example` for available options.

## Stellar Integration

This template uses the Stellar SDK to interact with the Stellar network. The network configuration (mainnet/testnet) is set during deployment.

## Customization

This template is designed to be customized through the CRAFT platform. All branding, features, and network settings are injected during the deployment process.
