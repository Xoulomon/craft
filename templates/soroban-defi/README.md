# Soroban DeFi Template

A DeFi application template built on Stellar's Soroban smart contract platform.

## Features

- Smart contract interactions
- Liquidity pools
- Yield farming
- Wallet integration
- Customizable branding

## Getting Started

```bash
npm install
npm run dev
```

## Soroban Integration

This template uses Soroban RPC to interact with smart contracts on the Stellar network.
